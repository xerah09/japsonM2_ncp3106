Skill Forge

Project Description

Android Studio is an integrated development environment (IDE) created especially for Android development, and it was used to work on this project. It was my responsibility to ensure that the app is aesthetically pleasing, functional, and user-friendly in both its coding and design. This project's main goal was to develop an easy-to-use interface that offers seamless navigation and a fun user experience. I used a variety of technologies, such as XML to design the user interface elements and Java/Kotlin for the backend logic.
